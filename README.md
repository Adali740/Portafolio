## Sincronizador Multimedia en Tiempo Real
*Proyecto de solución de conectividad para entorno comunitario.*

Este proyecto nació para resolver un problema logístico en una iglesia local: la falta de cableado físico para conectar el audio (consola) con el video (datashow).

* **Tecnologías:** Node.js, Socket.io, HTML5/CSS3.
* **Logro Técnico:** Implementé una arquitectura **Servidor-Cliente** mediante WebSockets. El sistema permite que un dispositivo móvil controle la reproducción de video en una PC remota de forma sincronizada vía Wi-Fi.
* **Funcionalidades:** Reproducción sincronizada, control de eventos en tiempo real (`io.emit`), y manejo de estados del reproductor.

## Sagar - Catálogo Web Dinámico
*Sistema de gestión de productos con persistencia en la nube.*

Un sitio web desarrollado para la visualización de inventarios, enfocado en una experiencia de usuario fluida y carga eficiente de archivos multimedia.

* **Tecnologías:** Python (Flask), JavaScript, Supabase, SQL Server.
* **Reto Superado:** Gestión de almacenamiento de imágenes en la nube. Implementé la integración con **Supabase Storage**, manejando la persistencia de datos y la renderización dinámica mediante la recuperación de URLs firmadas.
* **Estado:** Terminado y desplegado en Render.

*Optimización y despliegue de modelos de lenguaje en hardware personal.*

Investigación activa sobre cómo hacer que la Inteligencia Artificial sea accesible y privada sin depender de la nube.

* **Herramientas:** Ollama, Llama.cpp, Aider.
* **Modelos Manejados:** Phi-3.5 (Microsoft), Qwen 2.5 (Alibaba), Llama 3 (Meta).
* **Habilidad Clave (Cuantización):** Experiencia optimizando el consumo de recursos, logrando reducir el uso de memoria RAM de modelos de **30GB a menos de 8GB** mediante técnicas de cuantización, permitiendo su ejecución fluida en laptops estándar.
